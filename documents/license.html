Преамбула

Настоящее лицензионное соглашение (далее — «Соглашение») заключается между вами (физическим лицом или организацией, именуемым «Пользователь») и RPText Game Bot (далее — «Сервис»), регулирует условия использования телеграм-бота для генерации текстовых игр.

1. Определения

  • Бот — телеграм-бот RPText Game Bot, предназначенный для генерации текстовых игр в различных жанрах.
  • Кристаллы — виртуальная валюта, используемая для активации функций Бота.
  • Лицензия — право на использование Бота в рамках выбранного пакета Кристаллов.

2. Предмет соглашения

  1. Сервис предоставляет Пользователю неисключительную, не подлежащую передаче лицензию на использование Бота.
  2. Использование Бота не ограничено по времени, однако каждая операция генерации игры требует расхода Кристаллов.
  3. Лимит Кристаллов определяется выбранным Пакетом лицензии.

4. Условия оплаты

  1. Оплата пакетов Кристаллов производится в рублях через платёжные системы, доступные в Telegram.
  2. Стоимость пакетов не подлежит возврату.

5. Права и обязанности сторон

Сервис обязуется:

  • Обеспечивать работоспособность Бота (за исключением форс-мажорных обстоятельств).
  • Предоставлять техническую поддержку через Telegram-чат.

Пользователь обязуется:

  • Не использовать Бот для генерации незаконного, оскорбительного или вредоносного контента.
  • Не декомпилировать, не изменять и не распространять код Бота.
  • Не перепродавать доступ к Боту или Кристаллы.

6. Ограничения использования

  1. Бот предназначен исключительно для личного использования.
  2. Коммерческое использование возможно только с письменного согласия Сервиса.

7. Срок действия и расторжение

  1. Лицензия действует бессрочно, если Пользователь не нарушает условия Соглашения.
  2. Сервис вправе заблокировать доступ к Боту при нарушении Пользователем условий Соглашения.
  3. Лицензия автоматически прекращается при полном расходовании Кристаллов или закрытии Сервиса.

8. Ответственность сторон

  1. Сервис не несёт ответственности за невозможность использования Бота из-за действий третьих лиц, сбоев в интернете или Telegram.
  2. Максимальная ответственность Сервиса ограничена суммой, уплаченной Пользователем за последнюю приобретенную лицензию.

9. Конфиденциальность

Сервис собирает и обрабатывает данные Пользователя в соответствии с Политикой конфиденциальности.
